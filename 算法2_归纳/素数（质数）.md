### 质数筛法

###### 1. 素数判断方法

判断x是否为素数，如果是，返回true，否则返回false

**原理**：素数定义

```c++
bool isprime1(int x) {
    if(x == 2) return true;
    if(x % 2 == 0 || x == 1) return false;
    for(int i = 3; i <= x / i; i += 2) {
        if(x % i == 0) {
            return false;
        }
    }
    return true;
}
```
